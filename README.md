# wszf's Skills

My collection of custom Claude Code skills for productivity and automation.

## Skills

| Skill | Description |
|-------|-------------|
| [migrate-config](./migrate-config/) | 导出/导入 Claude Code 完整配置，一键在新机器上复现环境 |
| [permission-manager](./permission-manager/) | 分析 skill 权限需求、扫描项目构建工具、一键授权 |

## Installation

```bash
# 添加 marketplace
/plugin marketplace add wszf/skills

# 安装 skill
/plugin install migrate-config@wszf-skills
```
